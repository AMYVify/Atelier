# Atelier

Steps to create a new branch for work:
Switch to main branch to update data:
    “Git checkout main”

Pull data
    “git pull”

Create a NEW branch
    “git checkout -b branchname main”

*After committing and work*
Push to new branch on GitHub
    “git push -u origin branchname”
